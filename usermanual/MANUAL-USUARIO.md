## Introducción

### 📌 Propósito del Documento

El presente manual de usuario tiene como objetivo proporcionar una guía clara, práctica y detallada para el uso y administración del sitio web **Winnie Pooh Jardín Infantil** (winniepoohjardininfantil.com).

Este manual busca facilitar la comprensión de la estructura y funcionamiento tanto del panel de administración como de la página pública, asegurando un uso correcto, eficiente y seguro de la plataforma.

### 👥 ¿A quién va dirigido?

Este documento está dirigido a los **responsables de la gestión del contenido** del sitio, permitiéndoles realizar de manera autónoma tareas como:
- Actualización de información
- Carga de imágenes y videos
- Administración general del sistema
- Sin necesidad de conocimientos técnicos avanzados

### 🌐 ¿Qué es la web y para qué sirve?

El sitio web de Winnie Pooh Jardín Infantil es una **plataforma digital** diseñada para presentar de manera clara, atractiva y organizada toda la información relevante del jardín infantil.

Se trata de un sitio web tipo **landing page extendida**, estructurado en diferentes secciones y páginas que permiten al usuario navegar de forma intuitiva y acceder fácilmente a contenidos como:
- Información institucional
- Sedes
- Servicios
- Galería multimedia
- Reseñas
- Medios de contacto

**Su principal objetivo es:**
- ✅ Brindar información completa a padres de familia interesados
- ✅ Generar confianza a través de contenido visual y testimonios
- ✅ Facilitar el contacto directo mediante canales como WhatsApp
- ✅ Servir como canal oficial de presencia digital del jardín

Además, el sitio cuenta con un **panel de administración** que permite actualizar el contenido de manera sencilla, garantizando que la información se mantenga siempre vigente sin necesidad de conocimientos técnicos.

### 🛠️ Tecnologías Utilizadas

El sitio web ha sido desarrollado utilizando tecnologías modernas que garantizan alto rendimiento, seguridad y una experiencia de usuario fluida.

#### Desarrollo y Estructura
- **React y Next.js**: Permiten construir una interfaz rápida, optimizada y adaptable a diferentes dispositivos
- **TypeScript**: Mejora la estabilidad del sistema al reducir errores durante el desarrollo

#### Diseño e Interfaz
- **CSS**: Encargado del diseño visual del sitio
- **shadcn/ui y Magic UI**: Proporcionan componentes modernos y consistentes
- **Framer Motion**: Añade animaciones suaves que mejoran la experiencia del usuario
- **Lucide React**: Utilizado para iconografía clara y estética

#### Infraestructura y Despliegue
- **Vercel**: Plataforma donde se encuentra alojado el sitio web, optimizada para alto rendimiento
- **GitHub**: Repositorio donde se gestiona el código fuente del proyecto
- **Supabase**: Sistema que gestiona la base de datos y almacenamiento de contenido
- **Hostinger**: Proveedor del dominio y servicios de DNS asociados

#### Funcionalidades Adicionales
- **Google Maps**: Integración para mostrar la ubicación de las sedes
- **Enrutamiento optimizado**: Navegación entre páginas sin recargas

## Gestión de Contenido

La gestión de contenido permite mantener el sitio actualizado mediante la administración de archivos multimedia (imágenes y videos).

Desde esta sección se pueden realizar acciones como:
- ✅ Subir contenido nuevo
- ✅ Organizar el orden de visualización
- ✅ Eliminar archivos existentes
- ✅ Actualizar elementos visuales clave

### 📤 Cómo subir imágenes y videos a la galería

Para agregar contenido a la galería:

1. **Acceder al panel de control**

**(CAPTURA: Vista del panel de control principal)**

2. **Hacer clic en el área "Sube fotos y videos"** o en el botón de carga

**(CAPTURA: Área de carga de archivos resaltada)**

3. **Seleccionar uno o varios archivos desde el dispositivo**

**(CAPTURA: Explorador de archivos del sistema operativo)**

4. **Presionar "Abrir"**

**(CAPTURA: Botón "Abrir" en el explorador de archivos)**

✅ **Resultado esperado:**
Los archivos se mostrarán automáticamente en la galería del sitio web.

**(CAPTURA: Galería con los nuevos archivos subidos)**

⚠️ **Recomendaciones:**
- Usar imágenes en buena calidad (pero optimizadas, es decir, no de mucho peso, para asegurar una carga más rápida)
- Evitar nombres de archivo con caracteres especiales
- Verificar que el contenido sea adecuado antes de subirlo
- Tamaño recomendado: menos de 10MB por archivo

### 🔄 Drag & Drop para reordenar

El sistema permite reorganizar el orden de las imágenes fácilmente mediante la funcionalidad de **arrastrar y soltar**.

**Para hacerlo, solo debe seguir los siguientes pasos:**

1. **Ubicar el archivo que se desea mover**

**(CAPTURA: Tarjeta de imagen en la galería)**

2. **Hacer clic sostenido sobre el ícono de agarre (⠿)** del elemento

**(CAPTURA: Cursor sobre el ícono de agarre)**

3. **Arrastrarlo a la posición deseada**

**(CAPTURA: Imagen siendo arrastrada con efecto visual)**

4. **Soltar para confirmar el cambio**

**(CAPTURA: Imagen en su nueva posición)**

✅ **Resultado esperado:**
El nuevo orden se reflejará automáticamente en la página web.

**(CAPTURA: Galería con el nuevo orden aplicado)**

💡 **Nota:** Los cambios se guardan automáticamente. Verás un mensaje "Guardando..." en la parte superior mientras se aplican los cambios.

Las imágenes de portada son elementos visuales principales de la galería. **Las primeras 6 imágenes** son las que se muestran en la vista previa de la landing page principal.

**(CAPTURA: Sección de galería en la landing page mostrando las 6 primeras imágenes)**

**Para actualizarlas:**

1. **Acceder al panel de administrador**

2. **Seleccionar la imagen que se desea poner en la portada**

**(CAPTURA: Imagen seleccionada en el panel)**

**(CAPTURA: Tarjetas con etiqueta de portada resaltadas)**

✅ **Resultado esperado:**

Para eliminar contenido:

## Problemas Comunes

### 🚫 No puedo iniciar sesión

**Posibles causas:**
- Usuario o contraseña incorrectos
- Error al escribir (mayúsculas, espacios, etc.)
- Sesión expirada

**Soluciones:**

1. Verificar que los datos ingresados sean correctos
2. Revisar que no haya espacios adicionales al copiar/pegar
3. Intentar escribir manualmente las credenciales
4. Recargar la página e intentar nuevamente

Si el problema persiste, contactar al soporte para restablecer el acceso.

### 📤 No suben las imágenes

**Posibles causas:**
- Conexión a internet inestable
- Archivo demasiado pesado
- Formato de imagen no compatible

**Soluciones:**

1. Verificar la conexión a internet
2. Intentar subir una imagen más liviana (recomendado: menos de 10MB)
3. Usar formatos comunes como `.jpg`, `.png` o `.webp`
4. Recargar la página e intentar nuevamente

💡 **Recomendación:** usar imágenes optimizadas mejora la velocidad del sitio.

### 🌐 La web no carga

**Posibles causas:**
- Problema de conexión a internet
- Caída temporal del servicio
- Problema del navegador

**Soluciones:**

1. Verificar la conexión a internet
2. Recargar la página (Ctrl + F5 o Cmd + Shift + R)
3. Intentar desde otro navegador (como Google Chrome)
4. Intentar desde otro dispositivo

Si el problema continúa, puede tratarse de un inconveniente técnico que requiere revisión.

### 🔄 Los cambios no se reflejan en la página

**Posibles causas:**
- Caché del navegador
- Los cambios aún no se han guardado
- Problema de sincronización

**Soluciones:**

1. Hacer clic en "Ver página en vivo" para verificar
2. Limpiar la caché del navegador (Ctrl + Shift + Delete)
3. Recargar la página con Ctrl + F5
4. Esperar unos minutos y volver a verificar

### 🗑️ Eliminé un archivo por error

**Solución:**

Lamentablemente, la eliminación de archivos es **permanente** y no se puede deshacer desde el panel.

**Opciones:**
1. Volver a subir el archivo si lo tienes guardado
2. Contactar al soporte si necesitas recuperar el archivo (puede requerir acceso a backups)

⚠️ **Recomendación:** Siempre confirmar antes de eliminar archivos importantes.
